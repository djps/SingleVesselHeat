subroutine which generates heat source from file (.cpp)

detail: In this routine, I load up the heating profile from the file heatFileName (stored as text file, with nx, ny, nz, maxValue as header, then x,y,z,heat, with x,y,z integers >= 0 and < nx,ny,nz). Heating_parameters also has a value that is the maximum heat deposition value to which we must rescale the acoustic profile. We also reduce absorption over the blood vessel. Finally, heating_parameters will also have 4 more parameters, two to determine when it starts switching on and how long until it’s going, and another two to switch it off. But these aren’t used in this routine. So num_heat_pars should be 5, but we only use 1. (.cpp)
